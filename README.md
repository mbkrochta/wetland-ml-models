# wetland-ml-models

Machine learning and statistical model comparison for wetland temperature prediction



This repository contains scripts, data structures, and model outputs used to analyze and predict thermal dynamics in wetland systems. The goal is to support research on how wetland attributes and climate variables influence water temperature using machine learning approaches.



\## Repository Structure



\- `data/` – Raw and processed data files (e.g., CSVs, shapefiles, NetCDFs)

\- `figures/` – Plots, diagrams, and visualizations

\- `models/` – Saved model objects (e.g., .rds, .pkl)

\- `notebooks/` – Jupyter or R Markdown notebooks for exploratory and model development work

\- `results/` – Model output summaries and evaluation metrics

\- `scripts/` – Core R or Python scripts for preprocessing, modeling, and analysis
